# Estratégia de Reestruturação de Títulos e SEO para Blog

## 1. FRAMEWORK DE OTIMIZAÇÃO

### Princípios Fundamentais
- **CTR (Click-Through Rate)**: Títulos que despertam curiosidade sem clickbait
- **Autoridade**: Linguagem técnica e perspectiva diferenciada
- **Ética**: Sem promessas falsas ou infração de código profissional
- **Engajamento**: Corpo do texto com linkagem interna e keywords naturais
- **Dwell Time**: Conteúdo profundo que mantém o leitor engajado

### Estrutura de Duas Camadas

```
<title> (Google): Máx 60 caracteres, palavra-chave no início
<h1> (Página): Versão completa, emocional e magnética
```

---

## 2. PADRÕES DE TÍTULOS POR TIPO

### Padrão A: Pergunta Provocativa + Resolução
**Fórmula**: "É [X] ou [Y]? Como [verbo] realmente."
- **Exemplo**: "É intuição ou ansiedade? Como diferenciar sinais genuínos."
- **Google Title** (58 chars): "É intuição ou ansiedade? Saiba como diferenciar."
- **H1 Completo**: "É intuição ou ansiedade? Como diferenciar sinais genuínos de alarmes falsos."
- **Por quê**: Pergunta resolve dúvida comum + especificidade técnica

### Padrão B: Quebra de Padrão + Conceito Técnico
**Fórmula**: "[Negação comum]: [Conceito real]"
- **Exemplo**: "Vibrar alto não cura traumas: Positividade tóxica na espiritualidade"
- **Google Title** (60 chars): "Vibrar alto não cura: Positividade tóxica explicada"
- **H1 Completo**: "Por que 'vibrar alto' não cura traumas: O perigo da positividade tóxica na espiritualidade"
- **Por quê**: Desafia crença comum + oferece perspectiva diferenciada

### Padrão C: Conceito + Impacto Clínico
**Fórmula**: "[Conceito]: Como [impacto] na [área de vida]"
- **Exemplo**: "Narcisismo encoberto: Como a falsa humildade sabota relacionamentos"
- **Google Title** (60 chars): "Narcisismo encoberto: Falsa humildade sabota relacionamentos"
- **H1 Completo**: "Narcisismo encoberto: Como a falsa humildade sabota seus relacionamentos"
- **Por quê**: Conceito específico + aplicação prática

### Padrão D: Síndrome/Fenômeno + Raiz Profunda
**Fórmula**: "[Síndrome]: A [raiz psicológica] que ninguém fala"
- **Exemplo**: "Síndrome do impostor: A raiz genealógica que ninguém fala"
- **Google Title** (60 chars): "Síndrome do impostor: Raiz genealógica desvendada"
- **H1 Completo**: "Síndrome do impostor em terapeutas: A raiz genealógica que ninguém fala"
- **Por quê**: Promessa de insight novo + profundidade clínica

### Padrão E: Ciclo/Padrão + Interrupção
**Fórmula**: "Por que você [padrão]? A lógica por trás de [comportamento]"
- **Exemplo**: "Por que você se sabota? A lógica por trás da autossabotagem"
- **Google Title** (60 chars): "Por que você se sabota? Lógica da autossabotagem"
- **H1 Completo**: "Por que você se sabota? A lógica psicológica por trás da autossabotagem"
- **Por quê**: Pergunta pessoal + explicação científica

## 3. OTIMIZAÇÃO DO CORPO DO TEXTO

### Estrutura SEO Interna
1. **Primeiro parágrafo**: Defina o problema + por que importa (50-80 palavras)
2. **Subtítulos H2**: Máx 70 caracteres, contêm keywords secundárias
3. **Linkagem interna**: 3-5 links para outros posts (nunca repetir mesmo texto)
4. **Autoridade**: Cite frameworks (Jung, Foucault, etc.) com contexto
5. **Call-to-action**: No final, com WhatsApp personalizado

### Checklist de SEO para Corpo
- [ ] Palavra-chave principal aparece em H1, primeiros 100 palavras, e 1-2x no corpo
- [ ] Palavras-chave secundárias naturalmente distribuídas
- [ ] Mínimo 3 links internos para posts relacionados
- [ ] Mínimo 1 referência a framework filosófico/psicológico
- [ ] Meta description: 150-160 caracteres, inclui palavra-chave
- [ ] Imagem com alt text descritivo
- [ ] Tempo de leitura estimado (visível para o leitor)

## 5. IMPLEMENTAÇÃO TÉCNICA

### Padrão de Código para useEffect

```typescript
useEffect(() => {
  // Versão curta para Google (máx 60 caracteres)
  document.title = 'Por que você se sabota? Lógica da autossabotagem | Jorge Dias';
  
  // Meta description (150-160 caracteres)
  const metaDescription = document.querySelector('meta[name="description"]');
  if (metaDescription) {
    metaDescription.setAttribute(
      'content',
      'Entenda a lógica psicológica por trás da autossabotagem. Como padrões inconscientes sabotam seu sucesso e relacionamentos. Análise clínica com perspectiva filosófica.'
    );
  }
}, []);
```

### H1 Magnético no JSX

```typescript
<h1 className="text-4xl md:text-5xl font-display font-bold text-foreground mb-4">
  Por que você se sabota? A lógica psicológica profunda por trás da autossabotagem
</h1>
```

## 6. MÉTRICAS DE SUCESSO

- **CTR esperado**: 4-6% (vs média de 2-3% para psicologia)
- **Dwell time**: 2+ minutos (vs 1:30 média)
- **Bounce rate**: <40% (vs 50%+ média)
- **Linkagem interna**: 3-5 links por post
- **Autoridade**: Mínimo 1 referência teórica por post
